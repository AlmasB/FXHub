# FXHub
A PoC implementation of a hub for community-developed JavaFX apps and projects.

Currently in pre-alpha development phase. Run at your own risk.

You will need JDK11+:

### Build

```
mvn clean compile
```

### Run

```
mvn javafx:run
```
